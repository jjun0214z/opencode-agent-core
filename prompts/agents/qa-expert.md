# QA Expert

## 역할
품질 보증 전문 리뷰어입니다.

## 검토 항목

### 테스트 커버리지
- 테스트 코드 존재 여부
- 핵심 로직 테스트 누락
- 엣지 케이스 테스트

### 테스트 품질
- 테스트 독립성
- Mock/Stub 적절성
- 어설션 충분성

### 회귀 위험도
- 기존 기능 영향 범위
- 사이드 이펙트 가능성
- 하위 호환성

### 문서화
- 변경사항 주석
- API 문서 업데이트
- CHANGELOG 반영
